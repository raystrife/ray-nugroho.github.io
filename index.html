Video Games

Ever since I was little, I have always been invested in video games. I love to play as a character in the video game and experience his/her story from his/her perspective. It feels like I’m watching a movie, but I can interact with the world inside a video game. For this reason, I have always appreciated open-world games that give me freedom to explore their world.

In the future, I’m interested to design my own video games. In preparation for this, I follow news about new game releases and reviews of recently released games. By reading these reviews, I can learn the characteristic of a great video game and mistakes that I can avoid. Currently, I’m taking an online course regarding video game programming in the Unity engine. Based on these commitments, I’m confident that I can design a video game that is both unique and appealing to the audience.

Programming

I have always dreamed to work as a full-time software developer. To achieve my goal, I have been enrolling in various computer science courses that are provided by the University of Waterloo. The most important subject that I learned from these courses is the object-oriented design by using C++. Since I’m interested in website development, I enrol in an online course that teaches front-end development by using HTML, CSS, and JavaScript. In addition, the co-op program of the University of Waterloo allows me to experience professional software development process from various software companies.

One of my first project is a chess game in C++ with several unique features. My game allows the user to save the current state of the game and load that specific save file. This is a great feature for the user to practice different scenarios or undo their mistakes. In addition, there is a “setup” feature that allows the user to place any chess pieces anywhere on the board before the game begin. In case a user is having difficulty to find a human opponent, the game comes with a built-in computer opponent with four-level of difficulty.
